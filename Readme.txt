# udacity-project-2
# Tournament Results version 1.0 05/30/2015

GENERAL USAGE NOTES
-------------------
- This application can be used for building the logic behind a Swiss pairing tournament.
- There is no frontend interface included with this release

REQUIREMENTS
------------
- Python v 2.7.9 is required to run this application
- psql is required to run this application

RUNNING TOURNAMENT RESULTS
----------------------
Before running any of the included python scripts, the psql scripts in tournament.sql should be run to create the required tables.

This application includes a unit test file, tournament_test.py.  This file can be run with python to demonstrate the functions included in the application.  Each of the functions in tournament.py can be used to execute the various steps needed to set up a Swiss Pairing Tournament.
